# Mid Priority: Storage Management View

## Goal
Add a lightweight view for storage usage and cleanup of downloaded maps and articles.

## Why
The app already exposes total map storage in Home, but there is no direct way to inspect or manage that storage.

## Starting Points
- Home statistics
- Settings screen
- local map/article storage services

## Notes
- Start with visibility before deletion tooling if needed.
- Keep cleanup actions safe and explicit.
